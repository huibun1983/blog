# 欢迎来到我的博客

经过一番折腾，新的博客终于上线了！

## 为什么选择静态博客？

在尝试了多种博客方案后，我最终选择了 **Hugo + Decap CMS + Cloudflare Pages** 的组合：

| 方案 | 优点 | 缺点 |
|------|------|------|
| WordPress | 功能强大 | 慢，需要维护 |
| Medium | 简洁 | 没有自主权 |
| 静态博客 | 快，安全，免费 | 需要技术基础 |

### 选择这个方案的理由

1. **极速加载** - 纯静态 HTML，加载速度飞快
2. **零维护** - 部署后无需操心服务器
3. **安全** - 没有数据库，没有 PHP，安全漏洞少
4. **免费** - GitHub + Cloudflare Pages 完全免费
